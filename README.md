Luxe Vista Resort - Mobile App

A simple mobile application developed for Luxe Vista Resort to allow users to log in, view a splash screen, and book appointments.

Technologies Used

- Java (Android)
- SQLite (Local Database)

Features

- Splash Screen  
- User Login & Authentication  
- Appointment Booking System  

Author

*Janith DinuJaya*

Installation

1. Clone or download the repository.  
2. Open the project in Android Studio.  
3. Build and run on an Android device or emulator.
